# Firefox Shortcuts Add-on

This add-on allows the user to customize Firefox shortcuts. It also helps to
discover already existing shortcuts.

- Version: 0.2
- Date: 2011/03/18
- Screenshots: t.b.d.
- Official site: <http://timtaubert.de/firefox-shortcuts>
